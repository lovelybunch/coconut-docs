    Terminal

    Coconut's web-based terminal lets you run commands directly in your browser. It's perfect for development tasks like starting servers, running tests, and managing your project files.

    Why Use the Terminal?

    • Persistent sessions - Your npm run dev keeps running even if you close your browser
    • No setup required - Works immediately without any configuration
    • Always accessible - Available anywhere you can access your Coconut project

    How to Use

    Opening a Terminal

    Click the "Terminal" tab in your Coconut interface. That's it!

    Running Commands

    Use it just like your regular terminal:

    # Start your development server
npm run dev
 
# Install dependencies
npm install
 
# Run tests
npm test
 
# Use git
git status
git add .
git commit -m "Update feature"

    Common Tasks

    Development Servers

    # Start and leave running
npm run dev
# or
yarn dev
# or  
python manage.py runserver

    Your server stays running even if you close the browser!

    File Management

    # Create files and folders
touch README.md
mkdir src/components
 
# Copy and move files
cp file1.txt file2.txt
mv oldname.txt newname.txt
 
# View files
ls -la
cat package.json

    Git Workflow

    # Check status and add files
git status
git add .
 
# Commit and push
git commit -m "Add new feature"
git push origin main
 
# Pull latest changes
git pull

    Useful Tips

    Multiple Sessions

    Open multiple terminal tabs to run different commands simultaneously - like one for your dev server and another for git commands.

    Keyboard Shortcuts

    • Ctrl+C - Stop the current command
    • Ctrl+L - Clear the screen
    • Up/Down arrows - Browse previous commands
    • Tab - Auto-complete file and folder names

    Session Persistence

    Your terminal sessions stay alive for days, so long-running commands won't get interrupted if you:

    • Close your browser
    • Lose internet connection
    • Switch to a different tab

    The terminal automatically reconnects when you come back.

    Common Issues

    Command Won't Stop

    Press Ctrl+C to stop a running command.

    Terminal Seems Frozen

    Try pressing Enter or Ctrl+C. If that doesn't work, refresh the page.

    Connection Lost

    The terminal will automatically try to reconnect. If it doesn't work after a few seconds, refresh the page.

    Can't See My Files

    Make sure you're in the right directory:

    pwd              # Shows current directory
ls               # Lists files in current directory  
cd /path/to/project   # Navigate to your project
